# Cable Billing App

This project is currently **in progress** 🚧

It is a web application built using Next.js for managing cable billing and customer records.

---

## Status

⚠️ Work in Progress (WIP)
Features are still being developed and may change frequently.

---

## Tech Stack

- Next.js
- Node.js
- JavaScript
- Tailwind CSS
- SupaBase
- Vercel

---

## Getting Started

Install dependencies:

```bash
npm install
```

Run development server:
```bash
npm run dev
```
Open:

http://localhost:3000

Note

This project is not completed yet and is under active development.
